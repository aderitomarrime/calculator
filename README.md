## Overview
A website version of a calculator that performs basic mathematical operations such as addition, subtraction, multiplication, and division. The calculator supports floating-point numbers and allows the user to remove the last input, among other features.


## Live Demo

Visit the website: [Live Demo](https://aderitomarrime.github.io/calculator/)


## Features

- Minimal interface  
- Performs basic math operations (addition, subtraction, multiplication, and division)  
- Evaluates only one pair of numbers at a time  
- Rounds results with long decimals  
- Clear button to reset all input  
- Division by zero is not allowed  
- Supports floating-point numbers  
- Backspace button to remove the last input  
- Keyboard support

## Installation

To run this project locally, follow these steps:

1. Clone the repository:

```sh
git clone https://github.com/aderitomarrime/calculator.git
```

2. Navigate to the project 
```sh
cd calculator
```
3. Open index.html in your browser


## How to use

Once the website is open:

1. Digit a number.
2. Choose a mathematical operator (addition, subtraction, multiplication, or division).
3. Digit a second number.
4. Click the "=" button to see the result.
5. To clear all input and reset the calculator, click the "C" button.
6. If you make a mistake while typing, click the "Backspace" button to remove the last input.
